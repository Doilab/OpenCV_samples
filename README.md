# OpenCV_samples

## programs
- sample_data :<br/>
プログラムで扱う，画像データを保存しているディレクトリ．
- output_data :<br/>
出力された画像データの保存先．
- camera.py :<br/>

- imagefile.py :<br/>
    - src :<br/>
    読み込んだ画像と，マウスポインタのx,y座標，RGB値の表示．
    - dst :<br/>
    読み込んだ画像を上下反対にして表示．また，マウスポインタのx,y座標，RGB値の表示．

- edge_canny.py :<br/>
Canny法を用いた，エッジ検出プログラム．

- numPy_creation.py :<br/>
NumPy配列を用いた画像出力プログラム
- video_show.py :<br/>
動画ファイル操作のプログラム
